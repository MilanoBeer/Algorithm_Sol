```python
# BJ_2920_음계 
# c d e f g a b C : 1 2 3 4 5 6 7 8 

list = list(map(int, input().split()))
# print(list) 

if sorted(list) == list :
    # 오름차순 정렬된거랑 지금이랑 같으면
    print("ascending")
elif sorted(list, reverse=True) == list:
    print("descending")
else:
    print("mixed")
```

✅ sorted 함수!

- 새로운 리스트 리턴 O
- 내림차순 정렬 : `sorted(list_name, reverse=True)`
